# Code of Conduct

This project follows a simple and respectful code of conduct.

## Expected Behavior
- Be polite and respectful.
- Provide constructive feedback.
- Collaborate and help others when possible.

## Unacceptable Behavior
- Harassment or insults.
- Spam or intentionally unhelpful behaviour.
- Any discrimination.

## Reporting
If you need to report unacceptable behaviour, open an issue in the repository.
